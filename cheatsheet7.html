branch br2 -- git cheatsheet


Local Operations:
git init -- Convert a folder to a git repository.
git commit -a -m [message] -- Get a snapshot of changes for all modified files.
git checkout -b [newbranchname] -- Create a new branch and switch your working directory to it.
(equivalent to git-branch followed by git-checkout)
git add [file | --all] -- Add file(s) to staged area.
git commit -m [message] -- Get a snapshot of changes for all staged files.
git branch [branch] -- Create a branch.
git checkout [branch|commit] -- Set pointer (HEAD) to a branch or a commit.
git branch -f [branch] [commit] -- Move(force) a branch to a prior commit.
Remote Operations:
git clone [http://github/...] -- Copy a repository that is already on GitHub.
git remote -v -- Display remote references (verbose)
git push [remote] [branch] -- Send comitted changes to a GitHub repository
